/*************************TECLAS DEL EXÁMEN P5*****************************/

tecla 'm'		-> para cambiar de modo (modo puntos, líneas, relleno, ajedrez, luces flat, luces smooth)
tecla 'M'		-> para rotar en el eje Z de la primera cámara (perspectiva, la que aparece al ejecutar por defecto)
tecla 'o' U 'O' -> para cambiar el objeto (cubo, tetraedro, ply, cilindro, cono, esfera, objeto jerárquico, skybox práctica 4, cuadro práctica 4, molino para selección)
tecla 'c' o 'C' -> para cambiar la cámara activa (perspectiva, ortogonal, inclinada)


/*******************TECLAS DE PRÁCTICAS ANTERIORES A LA 5**********************/


tecla 'q' p 'Q' -> para salir del programa
tecla 'v' o 'V' -> para cambiar el modo de visualización (inmediato, diferido)
tecla 'p' o 'P' -> para cambiar al siguiente parámetro en el objeto jerárquico
tecla 'a' o 'A' -> para activar y desactivar las animaciones del objeto jerárquico (por defecto están desactivadas)
tecla 'Z'		-> para incrementar el valor del parámetro actual en el caso de que haya animaciones
tecla 'z'		-> para decrementar el valor del parámetro actual en el caso de que haya animaciones
tecla '<'		-> para decelerar el parámetro actual en las animaciones
tecla '>'		-> para acelerar el parámetro actual en las animaciones
tecla 'l' o 'L' -> para mover la luz magenta
tecla 't' o 'T' -> para cmabiar el material cuando hay luces activas (material azul, material amarillo)
